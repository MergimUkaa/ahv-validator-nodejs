

## Installation

Via npm

``` bash
$ npm install @mergimuka/ahv-validator
```

## Usage
```bash
const check = require('@mergimuka/ahv-validator');
const result = check('756.9217.0769.85')
```

## Result
```bash
result = {
    valid: true,
    length: 13,
    description: "some description"
}
```
